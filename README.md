# Library Management System

## Overview
A Java console-based Library Management System that demonstrates Object-Oriented Programming concepts.

## OOP Concepts Demonstrated
- **Encapsulation**: Private attributes with public getters/setters
- **Inheritance**: Book/Magazine extends LibraryItem, Member extends Person
- **Polymorphism**: Method overriding (displayInfo(), displayDetails())
- **Abstraction**: Abstract LibraryItem class with abstract methods

## Features
- Add/View Books and Magazines
- Register Members
- Borrow and Return items
- Search items by ID or title
- Track borrowed items with due dates
- Different membership types (Standard, Premium, Student)
